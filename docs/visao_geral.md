# 🛡️ Arsenal DEV: O Inventário Definitivo do Desenvolvedor

O **Arsenal DEV** é uma plataforma *open-source* projetada para ser o centro de comando de ferramentas para desenvolvedores. Mais do que um simples catálogo, a aplicação funciona como um curador estratégico de ecossistemas, onde profissionais podem descobrir, organizar e favoritar as melhores ferramentas para cada fase do ciclo de desenvolvimento.

A ideia central nasce da necessidade de combater a **"fadiga de decisão"** no mundo tech. Em vez de navegar por dezenas de sites, o desenvolvedor encontra em um só lugar o arsenal necessário para sua "batalha" diária — seja ele um especialista em Frontend, Backend, DevOps ou Mobile.

---

## 🚀 Funcionalidades de Destaque

*   **📦 Inventário por Camadas:** Classificação inteligente de ferramentas entre Frontend, Backend, Mobile, Design, QA e Infraestrutura.
*   **🛠️ Kits de Setup (Stacks):** Visualizações dinâmicas baseadas em tecnologias específicas. Ao selecionar "Python", o usuário recebe um guia completo: da linguagem aos frameworks, linters e bibliotecas de teste recomendadas.
*   **💰 Status de Custo:** Informação clara sobre o modelo de negócio (Open Source, Gratuito, Freemium ou Pago).
*   **⭐ Inventário Pessoal (Favoritos):** Autenticação via GitHub para salvar ferramentas prediletas e gerar links de acesso rápido.
*   **⚡ Guia Rápido de Instalação:** Instruções curtas ou comandos de terminal (ex: `npm install` ou `brew install`) para acelerar o setup do ambiente.

---

## 💻 The Arsenal Stack (Stack Tecnológica)

Para garantir modernidade e escalabilidade, a arquitetura utiliza as tecnologias mais robustas do ecossistema JavaScript/TypeScript.

| Camada | Tecnologia | Motivação |
| :--- | :--- | :--- |
| **Frontend** | `Next.js 14+` | Server Components para SEO agressivo e rotas dinâmicas. |
| **UI** | `Tailwind & Shadcn/UI` | Interface moderna, modo escuro nativo e consistência visual. |
| **Backend** | `Node.js & TypeScript` | Segurança de tipos e facilidade para novos contribuidores. |
| **Auth** | `NextAuth.js` | Integração nativa com GitHub para a comunidade dev. |
| **Banco** | `PostgreSQL & Prisma` | Relações complexas tratadas de forma simples e intuitiva. |
| **Infra** | `Docker` | Ambiente de desenvolvimento padronizado e isolado. |

---

## 🤝 Filosofia Open Source

O Arsenal DEV foi pensado **pela comunidade e para a comunidade**. A escolha desta stack visa baixar a barreira de entrada para novos desenvolvedores.

Qualquer pessoa pode ajudar a expandir o arsenal através de:
1.  Sugerindo novas ferramentas via PR ou Issue.
2.  Refinando setups de tecnologias emergentes.
3.  Contribuindo com o código core da plataforma.
